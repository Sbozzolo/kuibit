Reference on kuibit.utils¶

The utils module contains functions used throughout the codebase.

In particular: - get_logger() can be used to get kuibit’s own logger. - set_verbosity() modifies the format and output level for a given logger,

by default kuibit’s own logger.

kuibit.utils.get_logger(name='kuibit')[source]¶

Return the logger with the given name. By default, this returns kuibit’s own logger.

Returns

Logger.

Return type

logging.Logger

kuibit.utils.set_verbosity(level='INFO', logger=None)[source]¶

Set format and log level for the given logger.

Parameters

  • level (str) – Desired level of the logger (INFO or DEBUG).

  • logger (logging.Logger) – Logger.
