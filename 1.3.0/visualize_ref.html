Reference on kuibit.visualize¶

The visualize module provides functions to plot kuibit objects with matplotlib.

kuibit.visualize.setup_matplotlib()[source]¶

Setup matplotlib with some reasonable defaults for better plots.

Matplotlib behaves differently on different machines. With this, we make sure that we set all the relevant paramters that we care of.

This is highly opinionated.
